The Wire symbol is a standalone glyph derived from the logo lettering. It should be used as an icon to indicate Wire apps, preferably with the full product name added in text. The symbol can also be used on its own if space is tight and the logo needs to be shortened or presented in a compact way.

Like the logo, you should never misuse the symbol.

  • Don’t bend or twist
  • Don’t distort or stretch
  • Don’t rotate or tilt
  • Don’t use on a photo
  • Don’t use in multiple colors
  • Don’t apply effects

The Wire symbol can either be used as a freestanding element or within a surrounding shape. This approach can be used to create icon variants for different software platforms or app stores. When placing the symbol inside a shape, you should adhere to platform design guidelines.

Whether the symbol appears as a standalone element or in a shape, it’s best to use it in black or white. If you need a colored version, pick one of the Wire palette colors.
