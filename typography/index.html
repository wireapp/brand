The Wire product user interfaces and website use standard system fonts. This provides the best support for multiple languages, advanced typographic features and Unicode compatibility. On Apple operating systems, Wire uses San Francisco, on Android Roboto; and on Microsoft platforms Arial. If an operating system has defined a different default font family, use that.
